# SQL Analysis

This folder contains SQL queries used to perform business analysis on the supply chain dataset.

## Objectives

- Analyze demand patterns
- Evaluate inventory performance
- Measure promotion effectiveness
- Identify stockout risks
- Compare regional performance

## Key Analyses

### Category Performance

Identify top-performing categories.

### Regional Demand Analysis

Evaluate demand across regions.

### Promotion Impact

Measure the effectiveness of marketing campaigns.

### Seasonal Analysis

Understand seasonal demand fluctuations.

### Inventory Risk Detection

Identify stores with potential stockout issues.

## Output

SQL insights support both exploratory analysis and Power BI dashboard development.
