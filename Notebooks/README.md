# Notebooks Folder

This folder contains all Jupyter notebooks used during the analytics workflow.

## Notebooks

### 01_data_cleaning.ipynb

Tasks performed:

- Data loading
- Missing value treatment
- Duplicate removal
- Data type validation
- Data quality checks

### 02_eda.ipynb

Tasks performed:

- Demand analysis
- Category analysis
- Regional performance analysis
- Promotion impact analysis
- Seasonal trends

### 03_feature_engineering.ipynb

Tasks performed:

- Date-based feature creation
- Inventory metrics
- Pricing metrics
- Forecasting features

### 04_demand_forecasting.ipynb

Tasks performed:

- Model building
- Model evaluation
- Demand prediction
- Forecast comparison

## Purpose

The notebooks document the complete analytical workflow from raw data to business insights.
