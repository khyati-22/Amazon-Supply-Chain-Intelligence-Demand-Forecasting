# 📂 Data Directory

This folder contains all datasets used throughout the **Amazon Supply Chain Intelligence & Demand Forecasting** project. It includes the original dataset, cleaned dataset, engineered datasets, and machine learning output files generated during different stages of the analytics workflow.

The data stored in this directory serves as the foundation for SQL analysis, exploratory data analysis (EDA), machine learning model development, and Power BI dashboard creation.

---

# 📌 Folder Purpose

The primary objective of this directory is to maintain a structured and organized data pipeline throughout the project lifecycle.

The datasets are categorized based on their role in the project:

- Raw Source Data
- Cleaned & Processed Data
- Machine Learning Outputs
- Model Evaluation Files

This organization ensures reproducibility, transparency, and easier maintenance of the project.

# 📄 Dataset Description

## 1. raw_supply_chain.csv

This is the original dataset before any preprocessing.

It contains raw transactional supply chain information collected across multiple stores, products, and regions.

This dataset is preserved for reference and reproducibility purposes.

### Contents

- Store Information
- Product Information
- Inventory Details
- Pricing Information
- Sales Records
- External Business Factors
- Historical Demand

This file should remain unchanged.

---

## 2. clean_supply_chain.csv

This dataset was generated after performing data cleaning and feature engineering.

It serves as the primary dataset used throughout the remainder of the project.

The following operations were performed during preprocessing:

- Converted Date column into datetime format
- Verified missing values
- Removed duplicate records
- Corrected data types
- Standardized column names
- Created additional business features
- Prepared the dataset for SQL and Machine Learning

This file is used for:

- SQL Analysis
- Exploratory Data Analysis
- Machine Learning
- Power BI Dashboard

---

## 3. demand_predictions.csv

This file stores the prediction results generated by the final machine learning model.

Each record contains:

- Actual Demand
- Predicted Demand

This dataset is used for:

- Model Evaluation
- Prediction Comparison
- Power BI Forecast Dashboard

---

## 4. feature_importance.csv

This file contains the feature importance scores generated by the Random Forest model.

It helps identify which variables contribute most to demand prediction.

Typical important variables include:

- Units Sold
- Inventory Level
- Price
- Promotion
- Competitor Pricing
- Seasonality

This dataset supports business decision-making by highlighting key demand drivers.

---

## 5. model_metrics.csv

This file stores the evaluation results for all machine learning models developed during the project.

Typical metrics include:

- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

These metrics help compare model performance and justify the selection of the final forecasting model.
